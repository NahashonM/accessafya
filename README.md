# GraphQL API

GraphQL api to be consumed by the front-end team
## Usage
Use `npm run start` to start the server or `npm run dev-server` with nodemon 

#### Dependancies
* nodejs: 14.15.1
* express: ^4.17.1
* apollo-server-express: ^2.19.0
* graphql: ^15.4.0
* graphql-tools: ^7.0.2
* mongodb: ^3.6.3
* mongoose: ^5.11.3
